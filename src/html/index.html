  • Cabin Fever

    Demo Code

    App that allows users to list personal property, like a mountain cabin, for rent. Users search for rentals by city and property type. They view up-to-date property details, and follow a link to the proprietary booking agent to make reservations.


    Cabin Fever solves a problem specific to the North Georgia market. If a party searches for cabin rentals near Ellijay, Georgia, for example, they could search across a dozen or more independent sites in order to find a suitable property. The app allows users to search across these platforms in one easy step, making for a streamlined user experience.


  • Vote ATL

    Demo Code

    Vote securely online and follow live polling results in local elections with Vote ATL. It is designed to promote and increase civic participation in local government elections. Users register with a Georgia voter identification number, and are permitted access to the mobile polls during the same hours as physical polling locations. After voting, users can view live polling results right from their phone, wherever they happen to be.


  • Hint Pic

    Demo Code

    What happens when you cross SnapChat with Sudoku? Imagine an app that is not only social and visual, but adds a dose of humor, too: this is Hint Pic. Users upload photos with a hidden caption, which friends guess for points. Hint Pic is a way to share, connect and have fun. Users register and securely log-in, post pictures, play the game, and share with friends.
